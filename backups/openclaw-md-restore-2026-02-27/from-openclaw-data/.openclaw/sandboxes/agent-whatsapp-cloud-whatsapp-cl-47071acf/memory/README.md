# Memory Layout - WhatsApp Cloud

Struttura:
- `contacts/` profili per numero (`+<numero>.md`)
- `reports/` riepiloghi giornalieri (`YYYY-MM-DD.md`)
- `policies/` regole operative e sicurezza memoria
- `context/` profilo business e linee guida di messaging

Obiettivi:
- memoria minima ma utile alla gestione lead/clienti
- separazione netta tra dati cliente e policy
- nessun dato personale non necessario

Brand esterno:
- Nome pubblico assistente: `WellaNet Dev Assistant`
- Evitare riferimenti non necessari a naming interno legacy.
